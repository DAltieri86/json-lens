JSON Lens Privacy Policy

Last updated: May 18, 2026

JSON Lens is a Chrome extension that formats JSON responses directly in your browser. This policy explains what data the extension processes and how that data is handled.

Personal Data

JSON Lens does not collect, sell, rent, or share personal data. The extension does not create user profiles and does not use analytics or tracking scripts.

Page Content

JSON Lens reads the visible body text of pages only to detect and format raw JSON responses. The content of pages you visit is processed locally in your browser and is not transmitted to JSON Lens servers or third-party servers.

Payment Status

The only account-related data used by JSON Lens is payment status, such as whether the current user has access to Pro features. Payment handling is provided by ExtensionPay. ExtensionPay may process payment details and account email addresses according to its own privacy policy: https://extensionpay.com/privacy.

Local Preferences

User preferences such as theme, font size, behavior toggles, cached payment status, account email when provided by ExtensionPay, and upgrade banner dismissal time are stored locally through chrome.storage.local. These preferences remain on your device and are not transmitted by JSON Lens.

Permissions

  • <all_urls>: Required to detect and format JSON responses on any website.
  • storage: Required to save local preferences and cached payment status.
  • activeTab: Required to apply changed settings to the active JSON page.
  • scripting: Required to inject the JSON viewer into raw JSON response pages.
  • webRequest: Required to inspect response Content-Type headers for JSON detection.

Data Retention

JSON Lens does not retain page content. Local preferences remain in Chrome storage until you uninstall the extension or clear extension storage.

Contact

For privacy questions, contact the developer using the support email listed on the JSON Lens Chrome Web Store listing.
